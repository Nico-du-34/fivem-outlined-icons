# TILP FREE FiveM Icons

Ce dépôt contient une collection de définitions d'icônes et d'objets pour FiveM, organisée par catégories.
Chaque catégorie possède un fichier Lua principal et, pour certaines, un dossier complémentaire.

## Structure du dépôt

- `all_items.lua` : mapping global de tous les items.
- `farming.lua` : exemple de catégorie principale pour l'agriculture.

## Utilisation

1. Ouvrir le fichier principal d'une catégorie (`*.lua`).
2. Consulter les définitions et mappings d'icônes dans les fichiers correspondants.
3. Vérifier `all_items.lua` pour la liste globale de tous les items.

## Remarques

- Les fichiers et dossiers sont nommés par catégorie métier ou type d'objets.
- Le dossier `[licenses and ids]` semble correspondre à la catégorie `licenses_and_ids.lua`.
